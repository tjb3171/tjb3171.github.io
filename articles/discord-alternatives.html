A list of Discord Alternatives for non-super-nerdy people

Discord has announced that soon, all users in any country will have to verify their age using government ID or selfies.

While they claim that your data is quickly deleted, k-ID's privacy policy on this is very vague and they may send that data to even more 3rd parties, making it subject to their policies too. How quickly they delete it also isn't very clear.

A recent Discord data breach leaked at least 70,000 people's IDs because another 3rd party didn't handle data properly, which really helps us trust them.

If you don't comply, you can't:

  • Access channels and servers set as NSFW (why are you using Discord for that anyway)
  • Disable some safety settings (which are annoying but I can live with them)
  • Speak on stages (you can just use normal voice chats, with some downsides)

However, the precedent this sets (for worldwide age verification regardless of if your country's laws require it or not) will lead to more companies locking more things behind risking being doxxed in the future.



  1. Stoat (formerly known as Revolt)

    2. Simple to use: 10/10. Customisable: 8/10. Does everything you need: 7/10. Has Bots. Moderation: 9/10. Web, Windows, MacOS, Linux, iOS, iPadOS, Andriod. Overall: 6/10

    Stoat is very simple and is basically the same as Discord. Development on it is kinda slow and it lacks some features, with things like screenshare being a "coming soon". However it's super staightforward and easy to use.

    It's also quite different from Discord and might take some getting used to, but it gets the job done pretty well.

    If you want to use it, the download is here.



  2. Matrix (using the default server and Element client)

    3. Simple to use: 7/10. Customisable: 10/10. Does everything you need: 7/10. Has Bots. Moderation: 6/10 (use a bot for this). Web, Windows, MacOS, Linux, iOS, Andriod. Technical knowledge required if running a very large community. Overall: 7/10

    Matrix has a reputation for being confusing as heck, probably because a lot of youtubers are bad at explaining it, but I personally found it quite simple. You can do all sorts of crazy stuff and self-host and other confusing things, but you don't have to do any of that to use it as a messaging app.

    Matrix can be used with lots of different clients, each with different features, and you can self-host but you can also join a pre-existing server and use it like normal.

    It's also quite different from Discord and might take some getting used to, but it gets the job done pretty well. Keep in mind that while it's good for small group chats, moderation for bigger group chats and communities can get annoying.

    The element client is the most straightforward and easy-to-use one, you can download it here.



  3. Fluxer

    4. Simple to use: 9/10. Customisable: 8/10. Does everything you need: 9/10. Bots are not implemented yet. Moderation: 10/10. Web (works on Mobile), Windows, MacOS, Linux. Native mobile apps are WIP. In Beta; potentially slightly unstable. Overall: 9/10

    Basically the same as Discord; the UI is very similar and everything works pretty much the same. Some features missing, such as Discovery, but this is planned to be added soon.

    While in Beta, they manually review every registration which takes time so it takes a few days to create an account.

    The download is here and the Web-App is here.



  4. RootApp

    5. Simple to use: 9/10. Customisable: 6/10. Does everything you need: 9/10. Apps & Bots coming soon. Moderation: 9/10. Windows, MacOS, Linux, iOS, Andriod. In Beta; a bit buggy. Can Import server templates from Discord. Overall: 8/10

    Does more or less everything a normal discord server should do. Also, in the future you will be able to use apps/bots to make it super customisable. (not implemented yet)

    Their privacy policy is a bit vague and apparently they might collect a lot of data about you, although apparently there are plans to rewrite it to be less vauge (and hopefully less intrusive) in the future. So if you care about that sort of thing, check the latest news to see what their privacy policy currently says.

    You can get it here.
